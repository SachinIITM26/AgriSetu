# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please follow the procedure below to report it.

1. **Do not disclose the vulnerability publicly until it has been addressed.**
2. Send an email to [security@agrisetu.com](mailto:security@agrisetu.com) with the following information:
   - A description of the vulnerability
   - How to reproduce the issue, if applicable
   - Any proof of concept or evidence that can help us evaluate the reported security issue.
3. We will review your report and respond to you as soon as possible.
4. If the report is valid, we will work on a fix and notify you once it has been addressed.

## Security Measures
We take security seriously and use various measures to ensure the safety of our users and their data.

- Regularly update dependencies and libraries.
- Conduct code reviews and security audits.
- Encourage community contributions to identify potential security issues.
